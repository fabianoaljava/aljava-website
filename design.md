## 4. Colour Usage (Important)

### 60-30-10 Rule (Adjusted for Dark Mode)

* 60% → Dark backgrounds (#091722 / #0F2433)
* 30% → Primary blue (structure, headings)
* 10% → Teal + Amber (interaction, highlights)

### Rules

* Never use pure black (#000000)
* Avoid overly bright colours
* Keep contrast high for readability
* Use colour sparingly — let spacing do the work

## 5. Typography

### Font

* Primary: Atkinson / Inter

### Hierarchy

| Element | Style                   |
| ------- | ----------------------- |
| H1      | Large, bold, white      |
| H2      | Slightly smaller, white |
| H3      | Medium, white           |
| Body    | Light grey (#D9E2EC)    |

### Rules

* Strong hierarchy
* Short paragraphs
* Max width ~70 characters
* Avoid dense text blocks
